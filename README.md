# AirGesture Share

Share files with a hand gesture. No installation required.

## Features
- Multi-file sharing
- Gesture control (pinch to send)
- P2P transfer (100% private)
- Works on any device

## How to Use
1. Open app on both devices
2. Enter same room code
3. Select files
4. Pinch to send

## Tech
- Frontend: HTML, Tailwind, JavaScript
- Backend: Node.js, Express, Socket.IO
- P2P: WebRTC
